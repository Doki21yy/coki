🤖 Agent vs Skill

OpenClaw 核心概念 — 手机版

核心定义

Agent

身份 & 大脑
  • 独立身份（灵魂 SOUL.md）
  • 独立 session（对话历史）
  • 独立 workspace（文件系统）
  • 独立 auth（API 密钥）
  • 连接到 Channel（飞书等）
≈ 一个人
有自己的大脑、记忆、性格

Skill

能力 & 工具
  • 放在 workspace/skills/
  • SKILL.md + 脚本 + 参考文档
  • 定义"如何完成任务"
  • 可被多个 Agent 共享
  • 可包含 agents/ 子 agent
≈ 工具腰带里的技能卡
用时取出，多人可共用
